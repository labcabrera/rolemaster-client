Under development

This application provides a set of services for playing Rolemaster developed by https://ironcrown.com/. It consists of the following modules:
  • Character generation: allows the creation of characters, their customization and leveling up.
  • Strategic sessions: equivalent to what would be a campaign within which there can be different tactical sessions.
  • Tactical sessions: execution of actions that generally involve combat or maneuver situations.
Pending features:
  • Security
    • OAUth2 + JWT integration
    • Character/Session authorization model
  • Characters
    • Complete the list of training packages
    • Power point development and spell lists
    • Selecting customizable skills options
    • List of talents and flaws
    • Filtering universes
    • Allow custom races
    • Direct modifications with administrator's permission on characters
  • Tactical
    • Complete weapon tables
    • Complete critical tables
    • Dynamic maneuvers
    • Static maneuvers
    • Configuration for adjusting the scale of movement
    • In-combat item management
  • Strategic
    • Rest healing
    • Trade and equipment purchase
    • Table encounters
  • Items
    • Item customization
    • Combat item features like extra critics on hit
  • Spells
    • Practically everything
